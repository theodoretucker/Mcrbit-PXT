Lesson 01 - Beautiful Image

Learn It

In this lesson, you will learn how to make the micro:bit display an image on the LED screen. You will create the project show in the video below.

Code It

  • First, you need to open the PXT editor. It is a web based editor, which means that you can access it with any computer connected to the internet.
  • Using either Mozilla Firefox or Google Chrome, naviage to http://pxt.io/. Then click on "Start Coding".
  • You will be presented with the following interface:
  • The PXT (or Programming eXperience Toolkit) editor uses blocks, much like Scratch, which you may have used before.
  • Let's get on with writing our first program for the micro:bit. Right click in the code area (on the right side of the PXT editor, and click "Delete Blocks".
  • Then, from the "Basic" tab, drag out blocks to make the following:
  • Our first program is now complete! Let's test it.

Learn It

  • Now you have written your code, you need to transfer it to the micro:bit.
  • In order to do this, you need to convert your block editor program into 'machine language' which the micro:bit can understand.
  • Watch the following screencast to learn how to convert your code into machine language, and upload it to the micro:bit:

Save It

  • Following the instructions in the video, download the .hex file of this project.
  • Move it to your schoolwork directory (as instructed by your teacher) and name it as below:
    • microbit-lesson01a.hex

Test It

  • Again following the instructions in the video, flash the code file onto the micro:bit.
  • The micro:bit should behave as shown in the first video.
  • Did your code work? If so, well done! If not, look through the code to find any mistakes. You can find a code listing above.
  • Now move on to the challenges...

Challenges

  • Challenge 1:
    • Now show an new image that will display on the micro:bit.
    • The images should show one after each other.
    • Only the 'forever' and 'show leds' blocks are needed.

Finish

    Congratulations! You have now completed lesson 1. Go back to the Beginner index, and move on to the next lesson.
