About

This site is a one-man effort. However, I do owe thanks to the following people:

  • James Brooks (http://github.com/brks) for the CSS files he created for the VLE of Bourne Grammar School (http://www.bournetocode.com).
  • The micro:bit Foundation for the lessons in the now deprecated Microsoft Block Editor (https://www.microbit.co.uk/blocks/lessons) which form the basis for this course.

This course is intended for Year 7 students, and is almost a direct copy of the Block Editor lessons mentioned above, but ported to the new PXT (Programming eXperience Toolkit) editor. Learners should be introduced to the micro:bit before beginning this course. They must work through all activities (including the challenges) in all lessons, starting with lesson 1, or key points will be missed out. The main activities on each page are meant to be copied, but the challenges are independent learning tasks, thus, no code sample is provided to students. All code files created throughout the couse are avaliable for teachers to download (and import into the PXT editor) in the code-examples folder of the project's Github repository, linked below.
